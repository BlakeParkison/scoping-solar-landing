The problem

Solar project data often gets cleaned up too late.

Sales assumptions become design problems.

Missing MSP details, roof flags, battery expectations, or tie-in context can create redesigns and awkward customer conversations.

Survey notes are hard to standardize.

Different reps and surveyors can describe the same electrical condition in different ways, making review slower.

Battery scopes need clearer language.

Backup boundaries, MID equipment, landing points, and homeowner expectations need to be separated before design starts.

The solution

A guided scoping layer before plan design.

Scoping Solar is being built to help solar sales, survey, and design teams collect cleaner project data before plan design begins.

1

Capture the job details in a structured intake.

2

Flag the electrical, roof, structural, PV, ESS, and expectation items that need review.

3

Send cleaner scope context into the design process.

What it does

Purpose-built scoping tools for solar professionals.

Project intake

Collect core project details in one structured starting point.

Site survey guidance

Guide reps and surveyors toward the information design actually needs.

Solar + battery scoping

Capture PV, ESS, backup, and equipment assumptions before they become design gaps.

Electrical service information

Record MSP ratings, panel type, service conditions, and downstream load-center context.

Tie-in planning

Separate the equipment connection point from backup architecture and service-interface details.

Saved project workflows

Test local saved projects now, with cloud-backed project libraries planned next.

Mobile and desktop access

Use the workflow where solar teams actually work: in the field and at the desk.

Future cloud sync

Latest beta updates

Scoping Solar Beta Update - Smarter PV + ESS Scoping

The Scoping Solar beta now includes expanded residential module libraries, inverter/microinverter package selection, ESS package selection, first-pass inverter-to-ESS compatibility checks, inverter quantity logic, Tesla Powerwall 3 Expansion Unit safeguards, and adaptive backup guidance based on selected battery model and quantity.

Beta progress May 25, 2026
  • Added detailed module libraries for Qcells, Canadian Solar, JA Solar, Jinko Solar, and Hyundai Energy Solutions.
  • Added inverter/microinverter package selection.
  • Added ESS manufacturer/model selection.
  • Added compatibility logic for Tesla Powerwall 3, Enphase, SolarEdge, and Qcells ESS pairings.
  • Added inverter quantity logic, including microinverter quantity locked to module count.
  • Added adaptive backup guidance using usable kWh, continuous kW, ESS quantity, and backup type.
  • Continued improving Sales-to-Survey/Design handoff for residential PV/ESS projects.

Scoping Solar is still in private beta. Final designs require engineering review, current manufacturer requirements, AHJ/utility acceptance, and applicable funder or program confirmation.
